###################
CodeIgniter custom "Userguide Toc"
###################

This is an add-on for the Codeigniter 3.x documentation. It brings
you an alternative horizontal navigation as you know in the 
Codeigniter 2.x documentation.

************
Installation
************

Inside the generated documentation root folder, navigate to the 
folder \_static\js and replace the theme.js with this one. 

Now you will see a small icon ntext to the top right searchfield.
If you click this, you will get the alternative navigation. Click 
it again to close it again. The tool use a cookie to remember 
your decision.
